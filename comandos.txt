# comandos git

## Primeiro comando que deve ser usado.
- git init: é usado para criar um novo repositorio git. 
- git add: adiciona arquivo ao 'stage', (stage é o lugar onde o arquivo vai ficar antes dele estar pronto) [.] é usado para adicionar todos os arquivos onde o bash foi chamado ou onde ele estiver no momento.
- git rm --cached <file> : esse comando é usado para remover o arquivo do stage.
